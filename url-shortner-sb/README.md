# URL Shortener - Spring Boot

A scalable URL shortening service inspired by Bitly.

## Features
- Short URL generation using Base62 encoding
- HTTP 302 redirection
- JWT-based authentication
- Click tracking and analytics
- MySQL persistence using JPA

## Tech Stack
- Java
- Spring Boot
- Spring Security (JWT)
- React
- MySQL
- Maven

## How to Run

```bash
./mvnw spring-boot:run
